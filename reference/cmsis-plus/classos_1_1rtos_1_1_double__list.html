void os::rtos::Double_list::remove ( Double_list_links &  node)

Remove the node from the list.

Parameters
nodeReference to the list node to remove from the list.

If the list has more than one node, update the neighbours to point to each other, skipping the node.

For lists with only one node, simply clear the list.

For more robustness, to prevent unexpected accesses, the links in the removed node are nullified.
