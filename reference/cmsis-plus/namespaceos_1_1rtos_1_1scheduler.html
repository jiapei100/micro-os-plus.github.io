Detailed Description

Scheduler namespace.

The os::rtos::scheduler namespace groups scheduler types and functions.

Types

using os::rtos::scheduler::status_t = typedef bool

Type of a variable holding scheduler status codes.

Usually a boolean telling if the scheduler is locked or not, but for recursive locks it might also be a numeric counter.

void os::rtos::scheduler::unlock ( status_t  status)

Unlock the scheduler.

Parameters
[in]statusThe new status of the scheduler.
Returns
Nothing.

Actually restore the scheduler status based on the given parameter, usually returned by a lock(). This allows for embedded critical sections to preserve the locked status until the outer one completes and invokes unlock().

Warning
Cannot be invoked from Interrupt Service Routines.
