Detailed Description

RTOS namespace.

The os::rtos namespace groups all RTOS specific declarations, either directly or via nested namespaces.

using os::rtos::result_t = typedef uint32_t

Type of values returned by RTOS functions.

For error processing reasons, most CMSIS++ RTOS functions return a numeric result, which, according to POSIX, when the call was successful, must be 0 (result::ok) or an error code defined in <errno.h> otherwise.
